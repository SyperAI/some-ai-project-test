## Prerequisites ⚙️
1. **Install [Python](https://www.python.org/downloads/):** version 3.11 is recommended
2. **Install [Git](https://git-scm.com/install/)**

## How to Install 🚀
1. **Clone Repository:**<br>
    ```bash
    git clone 
    ```
2. **Open the created folder**
3. **For Windows:** Run `install.bat`
<br> **For other OS:** Run `python installer.py`

> [!NOTE]
> At the moment, access is provided exclusively upon request to trusted persons. If you are not like that, save the project and wait for news.
